- Fetches connected social accounts from Blotato API (`GET /users/me/accounts`)
- Posts directly to Blotato (`POST /posts`) for auto-distribution to Facebook/Instagram
- Shows live connection status per platform (green = connected, dim = disconnected)
- **Status: 0 accounts connected** — Meta accounts need to be linked inside Blotato dashboard first

- Naledi's 4-post queue (2 queued, 2 drafts)

- "Move to Queue" action for draft posts
- Real-time publish result feedback

### ✍️ Post Composer
- Quick templates: Wild Game Biltong drop, Rwanda Coffee China window, Father's Day, Subscription Box
- Platform toggles: Facebook 📘 / Instagram 📷
- Character counter (warns at 2000+)

### 📅 Content Calendar View
- July 2026 calendar grid
- Posts shown per day by platform and status
- Today highlighted in amber

### 🔗 Accounts View
- Shows Facebook + Instagram connection status
