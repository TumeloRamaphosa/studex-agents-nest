## What I Built

### Blotato Content Pipeline (`blotato-content-pipeline.js`)
A full Node.js content pipeline that connects Naledi's content output to Blotato's multi-platform API, with Facebook Graph API as a direct fallback.

**Location:** `/workspace/workflows/blotato-content-pipeline.js`

---

## What It Does

### Core Capabilities
1. **Blotato Accounts Sync** — Queries `GET /users/me/accounts` to list all connected social platforms
2. **Multi-Platform Posting** — Posts content to Facebook, Instagram, YouTube, LinkedIn, Threads, TikTok via Blotato
3. **Facebook Direct Fallback** — When Blotato post fails, automatically switches to Facebook Graph API
4. **Instagram Publishing** — Full container → publish flow via Graph API
5. **Content Templates** — 4 Studex-specific templates for fast caption generation

### Content Templates (ready to use)
| Template | Use Case |
|---|---|
| `productLaunch` | New product drops, pricing |
| `testimonial` | Customer quotes |
| `behindTheScenes` | Farm/provenance content |
| `promotion` | Deals, hampers, limited offers |
