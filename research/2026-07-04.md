## TOPIC 1: Voicebox — AI Voice Agent for Studex Customer Service

### What It Is
Voicebox (github.com/jamiepine/voicebox) is a **free, open-source, local-first AI voice studio**. Think ElevenLabs +WisprFlow combined — but runs entirely on the Orgo VM.

**Core capabilities:**
- Voice cloning from seconds of audio
- 7 TTS engines: Qwen3-TTS, LuxTTS, Chatterbox Multilingual, HumeAI TADA, Kokoro + others
- Built-in **MCP server** — connects directly to Claude Code, Cursor, and any MCP-aware AI agent
- Single tool call: `voicebox.speak` — agent talks to you in a cloned voice
- 23 languages supported
- Global dictation (Whisper STT) — speak, it types into any field
- Stories editor for multi-track audio

### Could It Power Studex Customer Service?
**Yes — here's the architecture:**

```
Customer calls WhatsApp/Skype
  → Voicebox captures audio (Whisper STT)
  → Routes to Claude via MCP
  → Claude processes query (order status, pricing, halal)
  → Voicebox speaks response in Tumelo's cloned voice
  → Customer hears a real human voice — not a bot
```

**Or simpler (text-first):**
```
Incoming WhatsApp message
  → n8n workflow detects keywords
  → Triggers Robusca/Charlie agent
  → Agent responds via WhatsApp
  → Voicebox optionally reads responses aloud to Tumelo
```

**Limitations:**
- Voicebox is a desktop app (Tauri/Rust) — needs GUI environment
- On Orgo VM, audio I/O is tricky (headless server)
- Real-time two-way voice calls require additional telephony (Twilio, etc.)

### Verdict: **FUTURE QUICK WIN**
Voicebox is powerful for Tumelo's personal productivity — clone his voice, have it read order alerts, dictation. Full two-way voice customer service requires Twilio + Voicebox integration on VM. **Flag for Build Hour.**
