Get the Bot Online — 3 Steps

The ADAM SMASHER Discord bot is fully built (983 lines of Python, 14 commands, 5 agents). To go live, add your credentials below.

1
Create Discord App
discord.com/developers → Applications → New Application → Name it "ADAM SMASHER"
2
Add Bot + Get Token
Bot tab → Reset Token → COPY IT NOW. Enable MESSAGE CONTENT INTENT.
3
Get Your User ID
Discord Settings → Advanced → Developer Mode ON. Right-click yourself → Copy User ID.
