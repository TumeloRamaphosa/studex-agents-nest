# Content Approval Protocol

## Workflow
1. Naledi generates content → saved to Airtable "Pending Approval"
2. Manager notified on WhatsApp with content preview
3. Manager approves/rejects with optional feedback
4. Approved content → scheduled automatically
5. Rejected → Naledi regenerates with feedback

## SLA
- Manager review within 4 hours
- Rejection feedback must be actionable
