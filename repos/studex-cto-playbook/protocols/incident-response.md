# Incident Response

## Severity Levels
- P1 (Critical): Full system down → immediate response
- P2 (High): Major feature broken → < 2 hours
- P3 (Medium): Minor feature → next business day

## Steps
1. Identify severity
2. Notify CTO immediately (WhatsApp)
3. Rollback if needed
4. Diagnose root cause
5. Fix and test
6. Post-mortem
