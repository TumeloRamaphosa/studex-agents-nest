## VII. TOKEN COST MANAGEMENT STRATEGY

### What Costs the Most Tokens
1. **Long context windows** — Reading huge files, long conversation history
2. **Image generation** — Each image ~1000-2000 tokens equivalent
3. **Video generation** — High cost, use sparingly
4. **Subagent spawning** — Each subagent has its own context
5. **Web research** — Multiple URL extractions per search task

### Token Reduction Rules (Always Follow)
- Default to **MiniMax** (free/cheap) for general chat
- Use **Gemini 2.0 Flash** for content (cheapest API cost)
- Use **GPT-4o-mini / Haiku** for simple tasks
- Reserve **Claude Opus / GPT-4o** for complex analysis only
- Use **Ollama local** for repetitive/bulk tasks (free once Tailscale up)
- **Compact context** regularly — let MaxClaw prune old history
- **File-based handoffs** — Write results to files, don't repeat in context
- **Parallel subagents** — Run tasks simultaneously, not sequentially
