# Studex Meat — Obsidian Second Brain

> AI-first knowledge base for the Studex Meat operating system. Structured for LLM reasoning and daily agentic use.

## Vault Structure

```
studex-obsidian/
├── 00-Core/           # Identity, brand, glossary
├── 01-Business/       # Strategy, goals, competitive landscape
├── 02-Agents/         # Full agent roster + mandates
├── 03-Products/       # SKU catalog, pricing, suppliers
├── 04-Content/        # Naledi's content engine
├── 05-Campaigns/      # Active + past campaigns
├── 06-Finance/        # P&L, margins, pricing strategy
├── 07-Operations/     # Shopify, TCG, fulfillment, Supabase
├── 08-Research/       # Market intel, competitors, trends
├── inbox/             # Unsorted — daily capture
└── daily/             # Rolling daily notes
```

## Core Principles
- Machine-readable frontmatter on every note
- Wiki-links between related notes (`[[note]]`)
- Updated by agents after every significant event
- Reviewed weekly (every Monday)
